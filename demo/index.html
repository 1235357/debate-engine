What is DebateEngine?

DebateEngine is a structured multi-agent system that uses multiple AI agents to analyze content from different perspectives, challenge dominant opinions, and reach evidence-based consensus.

Try it by entering your code, proposal, or text below!

🔍 System Architecture

Entry Layer

Python API | FastAPI REST | MCP Server

Orchestration Layer

QuickCritiqueEngine (sync) | DebateOrchestrator (async)

Provider Layer

LiteLLM (100+ providers) | 3 modes: stable, balanced, diverse

Schema Layer (Pydantic v2)

CritiqueSchema | ConsensusSchema | Config

Evaluation Layer

DebateEval: BDR, FAR, CV, CIS, CE, RD, HD

Output Layer

Markdown | SARIF | JSON

About the Demo

This demo simulates the core functionality of DebateEngine. When you submit content, it demonstrates how multiple AI agents work together to analyze, critique, and reach consensus.

To use the real DebateEngine in your project, install it with:

pip install debate-engine
